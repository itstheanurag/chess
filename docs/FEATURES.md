# Features

## Core Chess Features

- **Real-time multiplayer gameplay** with Socket.IO
- **Complete chess move validation** using chess.js
- **Support for all special moves**:
  - Castling (kingside and queenside)
  - En passant captures
  - Pawn promotion
- **Game over detection**:
  - Checkmate
  - Stalemate
  - Draw conditions (threefold repetition, fifty-move rule, insufficient material)
- **Interactive chessboard** with drag-and-drop piece movement

---

## User Features

- **User authentication** with Better Auth
- **User profiles** and account management
- **Leaderboard system** to track top players
- **Daily chess puzzles** to improve skills
- **Game statistics and history** tracking

---

## Game Management

- **Create and join games** with customizable settings
- **Game lobby** showing all available games
- **Live game previews** to spectate ongoing matches
- **Paginated game listings** for easy navigation
- **Game state persistence** with Redis caching for fast recovery

---

## Social Features

- **Real-time chat functionality** for player communication
- **Testimonials section** showcasing user experiences
- **FAQ section** for common questions and answers

---

## UI/UX

- **Modern, responsive design** with Tailwind CSS v4
- **Smooth animations** with Framer Motion
- **Interactive landing page** with multiple sections:
  - Hero section with compelling call-to-action
  - Features showcase
  - "How it works" guide
  - Live game previews
  - Daily puzzles section
- **Dashboard** with sidebar navigation for easy access
- **Toast notifications** for user feedback
- **Loading states** and comprehensive error handling
- **Mobile-responsive** design for play on any device
