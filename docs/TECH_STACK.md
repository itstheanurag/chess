# Tech Stack

## Backend

### Core Framework

- **Node.js** & **Express** — REST API server
- **Socket.IO** — Real-time bidirectional communication

### Database & ORM

- **Drizzle ORM** — Modern TypeScript ORM for database access
- **PostgreSQL** — Relational database (via pg driver)
- **Redis** — In-memory data store for caching and session management

### Chess Logic

- **chess.js** — Chess rules, move validation, and game state management

### Authentication & Security

- **Better Auth** — Modern authentication library
- **cookie-parser** — Cookie parsing middleware
- **cors** — Cross-Origin Resource Sharing configuration

### Validation & Error Handling

- **Zod** — TypeScript-first schema validation
- **http-errors** — HTTP error handling utilities

### Utilities

- **uuid** — Unique identifier generation
- **dotenv** — Environment variable management

### Development Tools

- **TypeScript** — Type-safe JavaScript
- **tsx** — TypeScript execution for development
- **ts-node** — TypeScript execution engine
- **drizzle-kit** — Drizzle ORM CLI tools

---

## Frontend

### Core Framework

- **React** (v19 with TypeScript) — UI library
- **Vite** — Fast development server and build tool

### State Management

- **Zustand** — Lightweight state management library

### Routing

- **React Router DOM** — Client-side routing

### Real-time & HTTP

- **Socket.IO Client** — Real-time updates from server
- **Axios** — Promise-based HTTP client

### Authentication

- **Better Auth** — Authentication client library

### Styling

- **Tailwind CSS** (v4) — Utility-first CSS framework
- **class-variance-authority** — Type-safe component variants
- **clsx** & **tailwind-merge** — Dynamic class name utilities

### Animations & Icons

- **Framer Motion** — Production-ready animation library
- **Lucide React** — Beautiful & consistent icon library
- **React Icons** — Additional icon library

### Chess Logic

- **chess.js** — Chess logic on the client side

### UI/UX

- **React Toastify** — Toast notifications
- **Motion** — Additional animation utilities

### Development Tools

- **TypeScript** — Type-safe JavaScript
- **ESLint** — Code linting
- **Vite Plugin React** — React support for Vite

---

## Development Environment

### Package Manager

- **pnpm** (recommended) or npm

### Version Control

- **Git** — Source control

### Code Quality

- **TypeScript** — Static type checking
- **ESLint** — Linting for both frontend and backend
